
# Automated Time Metric

A browser-based Customer Service Time Metrics Tracker built with HTML, CSS, and JavaScript.

## Features
- Weekly and monthly performance tracking
- Autosave with localStorage
- Excel export
- Backup and restore
- GitHub Pages compatible

## How to Use
1. Upload all files to a GitHub repository
2. Enable GitHub Pages (root directory)
3. Open `index.html`

No backend required.
